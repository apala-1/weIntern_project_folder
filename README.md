# Student Record System (CRUD Application)

This is a simple **Python CLI application** that allows users to **Create, Read, Update, and Delete (CRUD)** student records.  

This project uses **JSON** for persistent data storage and is designed for beginners learning Python file handling and data management.

## Tech Stack

- **Python**
- **JSON**

## Features

- **Create Student**
- **View Students**
- **Update Student**
- **Delete Student**
